# dva-generate
## install
npm install dva-generate -g
or
yarn global add dva-generate
## use in dva
- version
```
dva-gen -v
```
- list module
```
dva-gen -lm
```
- generate model file
```
dva-gen model modelName gatwayName
```
- generate module folder
```
dva-gen module modelName gatwayName moduleZhName
```
